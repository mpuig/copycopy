> Features

Optimized for developer workflow. Minimal latency.

Context Aware

Clipboard content determines available actions:

CONTENT        ACTION
-------        ------
URL            -> OPEN, SHORTEN
FILE           -> REVEAL, SHARE
IMAGE          -> CONVERT, OPTIMIZE
PLAIN_TEXT     -> TRANSFORM, SEARCH

History: NULL.
Local execution environment only.

User Scripts

Define custom macros using shell scripts. Inject clipboard content via {text} variable.

Native SwiftUI

Compiled binary size < 15MB. Memory footprint: Negligible. Framework: SwiftUI.
